# 🧩 Shortest Path Maze Solver

A Python project that finds the shortest path through a maze using the **Breadth-First Search (BFS)** algorithm.

The program reads a maze, finds the starting point (`O`) and the destination (`X`), then visualizes the shortest path step by step in the terminal using the `curses` library.

---

## 🚀 Features

- Finds the shortest path between two points.
- Uses Breadth-First Search (BFS).
- Visualizes the searching process in the terminal.
- Highlights the final path.
- Supports custom maze files.

---

## 🛠️ Technologies Used

- Python 🐍
- Queue
- Curses
- Breadth-First Search Algorithm (BFS)
